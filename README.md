# BasicDjangoApp
 Basic Django Blog management app

# Install PIP - Mac users

### Step1
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

### Step2
python3 get-pip.py

## Check pip version
pip3 --version

# Install pip env
sudo pip3 install pipenv


# Create virtual environment 
pipenv --python 3.8 install django
